🛡️ TruthSphere AI – AI-Powered Misinformation Detection and Trust Analysis for a Sustainable Digital Society.

🌍 SDG Alignment:-
    Primary SDG:* SDG 16 – Peace, Justice and Strong Institutions

📌 Project Overview:
     TruthSphere AI is an AI-powered misinformation detection and trust analysis system designed 
   to identify fake and real news using Machine Learning and Natural Language Processing.

📌 Problem Statement:-
       The rapid spread of misinformation through social media and digital platforms has made it difficult 
    for people to distinguish between reliable and fake information.
    How might we use AI to detect misinformation and provide trust analysis so that people can access reliable information
    and contribute to a more sustainable digital society?

🎯 Objectives:-
  🔍Detect fake and real news.
  👉Promote digital literacy.
  👉Encourage responsible information sharing.
  👉Support SDG 16.

🤖 AI Components Used:-
 🎯Machine Learning
 🎯Natural Language Processing (NLP)
 🎯TF-IDF Vectorization
 🎯Logistic Regression
 🎯Pattern Detection
 🎯Decision Support System.
